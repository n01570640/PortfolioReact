# Pharmacy Management System — Client

React frontend for the Pharmacy Management System. See the [project README](../../README.md) for the full overview, setup instructions, and architecture.

## Local commands

- `npm start` — run the dev server at `http://localhost:3000` (expects the API at the URL in `.env`)
- `npm test` — run the CRA test runner in watch mode
- `npm run build` — produce a production build in `build/`

This app was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).
